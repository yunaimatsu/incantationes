# INCANTATIONES
## Namespace
| Matchfile name | Prefix | Explanation |
|:---:|:---:|:---:|
|proofreading.yml|`pr:`|Each option determine the proofreading strictness|
|translation.yml|`tr:`|Each option determine the target/source languages|

## Prerequisite
It uses Espanso

## SETUP
> Archlinux
```sh
make espanso
make <category-name>
```
> Macbook
```sh
make espanso-mac
make <category-name>
```
